# 文档

Go编程语言是一个开源项目，旨在使程序员的工作效率更高。

Go语言表现力强、简明、干净、高效。它的并发机制使其能够轻松地编写程序，从而最大限度地利用多核和网络机器，而其新颖的类型系统能够实现灵活和模块化的程序构建。Go可以快速编译成机器代码，但又具有垃圾收集的便利性和运行时反射的力量。它是一种快速、静态类型化的编译语言，感觉就像动态类型化的解释语言。
