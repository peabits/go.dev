# 教程：Go入门

在本教程中，你将得到一个关于Go编程的简要介绍。在这一过程中，你将：

- 编写一些简单的“Hello, world”代码。
- 安装Go（如果你还没有安装Go）。
- 使用go命令来运行你的代码。
- 使用Go软件包发现工具来寻找你可以在自己的代码中使用的软件包。
- 调用外部模块的函数。
